# Void Scanner F-Droid Repository

This directory contains the F-Droid repository for Void Scanner.

## Files

- `index.xml` - F-Droid index (auto-generated by CI)
- `metadata/com.bubo.voidscanner.yml` - F-Droid application metadata
- `void-scanner-*.apk` - Signed release APKs (auto-deployed when pushed)
- `*.apk` (various sizes) - Icons and other assets
- `metadata/` - Additional app metadata
- `status/` - Build information

## Adding to F-Droid

Users can add this repository using the URL:

```
https://BuboTheWise.github.io/VoidScanner/fdroid/repo/index.xml
```

Or using GitHub Actions CI:

1. Update `gradle.properties` with new version
2. Commit and push to `main` branch
3. CI will automatically build APK and update F-Droid index
4. Add repository in F-Droid app: Settings → Add repository → Enter URL above
5. Refresh repository and search for "Void Scanner"

## Installation

After adding the repository, install Void Scanner in F-Droid:

1. Open F-Droid app
2. Go to Settings → Add repository
3. Enter: `https://BuboTheWise.github.io/VoidScanner/fdroid/repo/index.xml`
4. Trust repository
5. Refresh repository list
6. Search for "Void Scanner"
7. Install

## CI

Automatic builds and deployment via GitHub Actions `.github/workflows/fdroid.yml`:

1. Triggers on: push to main branch with changes to `fdroid/**`, `app/build.gradle`, or `gradle.properties`
2. Builds release APK
3. Copies APK to repo directory
4. Runs `fdroid update` to generate index
5. Commits and pushes changes
6. APKs deployed to GitHub Pages automatically

When CI succeeds, the new APK becomes available for download from F-Droid and manual APK installs.
