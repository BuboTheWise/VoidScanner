# How to Add Void Scanner to F-Droid

Void Scanner is available through F-Droid, a trusted source for free and open-source software.

## Manual Installation

**Option A: Add from Repository URL**
1. Open the F-Droid app on your Android device
2. Go to Settings → Add new repository
3. Enter repository URL: 
   ```
   https://github.com/BuboTheWise/VoidScanner
   ```
4. Tap OK to add

**Option B: Install from APK (Direct Download)**
1. Download the latest release APK from:
   - GitHub Releases: https://github.com/BuboTheWise/VoidScanner/releases
   - Ensure you get the `fdroid/repo/app-release-unsigned.apk` file

3. Enable "Unknown sources" in your device settings (Android 8.0+)
4. Open the downloaded APK and tap Install

## Automatic Updates

Once added, F-Droid will automatically:
- Check for new versions
- Download and install updates
- Keep your app updated without manual intervention

## Updating

If you already have Void Scanner installed via F-Droid:
- Open F-Droid app
- Go to Void Scanner
- Tap "Update" when available

## Requirements

- Android 10+ (API 29+)
- Internet connection (for initial F-Droid app update only)
- Wi-Fi and/or Bluetooth permissions

## Privacy

Void Scanner respects your privacy:
- All data is collected locally on your device
- No tracking or telemetry is sent
- SSIDs and BSSIDs are hashed in release builds
- Coordinates are obfuscated in release builds
